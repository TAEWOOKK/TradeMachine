## RSI부족·RSI과열이란?

매수 조건(골든크로스·상승추세·모멘텀)을 충족해도 **RSI 필터**에서 걸리면 매수하지 않는다.

| 사유 | 의미 | 조건 |
|------|------|------|
| **RSI부족** | RSI가 45 미만. 상승 모멘텀이 아직 약함 | RSI < 45 → 매수 보류 |
| **RSI과열** | RSI가 70 초과. 이미 과매수 구간 | RSI > 70 → 매수 보류 |

**예시 (2026-03-20 삼성전자)**  
- 골든크로스 조건은 충족했으나 RSI가 **44**여서 45 미만 → **RSI부족**으로 매수 보류  
- RSI가 45~70 구간으로 올라오면 다음 스캔에서 매수 검토

#### 실제 결과 (DB 기반)

| **3/12 말일 자산** | 9,999,870원 (보유 1종목) |
| **3/13 말일 자산** | 9,956,670원 |
| **누적 손익** | -43,200원 |
| **매수한 종목** | S-Oil(010950) 8주 @116,700원 — 상승추세 진입 |
| **익절/손절 꼴** | S-Oil 8주 116,700원 매수 → **손절** 111,300원 매도 (-43,200원) |
| **SKIP된 종목·사유** | (당시 RSI 30~70, 잔액부족 등) |

#### 발견된 문제

1. **10% 비율 → 고가주 1주도 못 삼**
   - 가용 760만원 × 10% = 76만원
   - SK하이닉스 1주 101만원 → 76만원 < 101만원 → 매수 불가
   - "잔액부족"으로 SKIP만 반복

2. **잔고 API가 0 반환**
   - 대시보드에는 전일 리포트로 760만원 표시
   - 매수 시 `get_account_summary()`가 0 반환 → "잔액부족" 발생

3. **RSI 30~45 구간 리스크**
   - 하락 추세가 아직 약할 수 있는 구간에서 진입 → 손실 가능성
